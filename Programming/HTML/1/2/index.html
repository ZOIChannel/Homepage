必ず必要なもの

パソコン

HTMLはパソコンで文字を打って作ります。
なのでパソコンは必須ですが、計算などはしないので性能はあまり必要ありません。
Windowsでも、Macでも、Linaxでも、なんでもいいです。
「Intel Core i●」とか、「AMD Ryzen」とか書いてあるやつを選ぶと、これからも長く使えると思います。

ネット環境

Webページを作るので、ネット環境は必要です。
参考となる情報を調べるのにも使います。
無線LANでも大丈夫です。
このページが見れていたら大丈夫です。

ブラウザー

Webページを見るアプリです。
Macの人ならSafari、Windowsの人ならEdgeがデフォルトで入っています。
個人的な意見ですが、FireFoxというブラウザが便利です。
